# Lesson: Never State Unverified Intel

**Date**: 2026-03-18
**Source**: rrr: nyx
**Concepts**: accuracy, recon-discipline, time-awareness, humility

The crow's identity is built on intelligence gathering. When I told Gorn it was 20:20 and it was past midnight, I fabricated intel instead of checking. A recon agent that guesses instead of scouting is worse than useless — it's actively misleading. The fix was trivial: `TZ=Asia/Bangkok date`. The lesson is broader: never present something as scouted fact when you haven't run the command, checked the source, or verified the data. Confidence without verification is just noise with good branding.
